# 画师

根据 [萌娘百科](https://zh.moegirl.org.cn/%E8%90%9D%E8%8E%89) 的定义：身高范围在130至160厘米之间，头身比在四至六之间，面相幼齿的女性角色即可归入萝莉范畴。

个人排除内容：

欧美画风、大部分都是黑白图的、画过萝莉但比例很少(如セネト)、孕肚(如夢双月)、巨乳太多(如毛玉牛乳)
